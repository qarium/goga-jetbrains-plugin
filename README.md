# Goga

The IntelliJ companion for the Goga workflow.

**Goga** helps teams move development to a higher level through package specifications described in `CODEMANIFEST`. This plugin makes that workflow feel natural inside JetBrains IDEs — with better navigation, less noise, and first-class `CODEMANIFEST` support.

## Why Goga

- Treat `CODEMANIFEST` files as YAML
- Keep Goga-driven projects easier to navigate
- Reduce Project View noise from supporting files
- Control visibility directly from the IDE

## Installation

Download the latest plugin ZIP from **GitHub Releases**, then install it from disk in your JetBrains IDE:

**Settings / Preferences → Plugins → ⚙️ → Install Plugin from Disk...**

## Compatibility

Built for IntelliJ Platform **2024.3+**.
